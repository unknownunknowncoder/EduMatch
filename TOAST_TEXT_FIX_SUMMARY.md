# 提示框文字显示问题修复总结

## 🐛 问题描述
用户反馈：发布经验成功后仍然没有文字提示，只有一个√符号，需要有文字提示："发布经验成功"

## 🔍 根本原因
在多个Vue组件的模板中，提示框的文字被错误地放在了 `sr-only` 类的span元素中：

```html
<!-- 错误的实现 -->
<div v-if="showMessage" :class="getMessageClasses(messageType)" v-html="getMessageIcon(messageType)"></div>
<span v-if="showMessage" class="sr-only">{{ messageText }}</span>
```

`sr-only` 类是Tailwind CSS的辅助类，用于屏幕阅读器可访问性，但对普通用户是视觉隐藏的。这导致用户只能看到图标（√），而看不到文字内容。

## ✅ 修复方案
将文字从 `sr-only` span中移出，直接放在提示框div内，与图标一起显示：

```html
<!-- 修复后的实现 -->
<div v-if="showMessage" :class="getMessageClasses(messageType)" class="flex items-center space-x-2">
  <span v-html="getMessageIcon(messageType)"></span>
  <span>{{ messageText }}</span>
</div>
```
