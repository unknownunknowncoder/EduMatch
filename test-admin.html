后台管理系统测试指南

访问方式

1. 确保您已登录系统

2. 使用管理员账户（email: admin@example.com）

3. 直接访问: /admin

功能模块

  • ✅ 用户管理 - 查看和管理所有用户
  • ✅ 内容管理 - 管理社区帖子
  • ✅ 学习计划管理 - 管理学习计划
  • ✅ 资源管理 - 管理学习资源

核心功能

  • 📊 实时数据统计
  • 🔍 搜索和筛选
  • ✏️ 批量操作
  • 👁️ 详情查看
  • 📄 分页显示

测试步骤

  1. 启动开发服务器: npm run dev
  2. 登录系统或创建管理员账户
  3. 访问 /admin 路由
  4. 测试各个管理模块的功能
  5. 检查数据加载、搜索、筛选、批量操作等

注意事项

  • ⚠️ 确保数据库连接正常
  • ⚠️ 需要有效的用户数据
  • ⚠️ 管理员权限检查基于邮箱或角色
  • ⚠️ 删除操作不可恢复
