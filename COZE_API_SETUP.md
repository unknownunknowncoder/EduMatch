# 扣子智能体API集成指南

## 🚀 功能概述

已成功将扣子智能体集成到EduMatch首页搜索功能，实现AI智能推荐教育资源。

## 📋 功能特性

### 1. 智能搜索
- 输入关键词，AI自动匹配B站和MOOC资源
- 60% B站高质量热门资源 + 40% MOOC系统课程
- 实时获取最新、热门的学习资源

### 2. 平衡推荐
- B站资源：播放量高、更新快、实操性强
- MOOC资源：系统性强、权威性高、证书价值
- 智能排序，优先推荐最适合的资源

### 3. 完整信息
- 每个推荐包含：平台、时长、学习数据、BV号等
- AI学习建议，制定个性化学习路径
- 一键跳转，直达学习资源

## 🔧 配置步骤

### 1. 获取扣子API凭证

#### 步骤1：创建扣子应用
1. 访问 [扣子开发者平台](https://coze.cn/)
2. 创建新应用或使用现有应用
3. 在应用设置中获取API Token

#### 步骤2：获取Bot ID
1. 在扣子平台找到你的智能体
2. 复制Bot ID（通常在智能体详情页面）

### 2. 配置环境变量

```bash
# 复制环境变量模板
cp .env.example .env.local

# 编辑配置文件
nano .env.local
```

填入实际配置：
```env
VITE_COZE_API_TOKEN=your_actual_coze_api_token
VITE_COZE_BOT_ID=your_actual_bot_id
```

### 3. 重启开发服务

```bash
# 停止当前服务
Ctrl + C

# 重新启动
npm run dev
```

## 🎯 使用方法

### 1. 首页搜索
1. 在首页搜索框输入关键词（如：线性代数、Python爬虫、3D动画）
2. 点击"AI推荐"按钮或按回车
3. 等待AI分析并返回推荐结果

### 2. 查看结果
- **最推荐**：AI认为最适合的资源，突出显示
- **其他推荐**：多个备选资源，包含B站和MOOC
- **学习建议**：AI生成的个性化学习路径

### 3. 开始学习
- 点击"立即学习"按钮，自动跳转到对应平台
- B站资源：新标签页打开视频页面
- MOOC资源：跳转到MOOC官网
