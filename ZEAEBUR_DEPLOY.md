## 🚀 部署步骤

### 1. 导入项目到 Zeabur
1. 登录 [Zeabur](https://zeabur.com)
2. 点击 "New Service"
3. 选择 "Deploy from Git"
4. 授权并选择你的 EduMatch 仓库

### 2. 验证部署模式
部署后，检查这些标志：
- ✅ **Docker 标志**：项目图标旁边有小鲸鱼
- ✅ **启动命令**：显示 `npm start`
- ✅ **Runtime**：显示 `Docker`
- ✅ **端口**：自动识别为 3014

### 3. 确认容器模式成功
查看 `Logs` 标签，应该看到：
```
🚀 EduMatch服务器启动成功！
📍 环境: production
🌐 地址: http://localhost:3014
```

## 🔧 环境变量设置
在 Zeabur 项目设置中添加：
```
COZE_API_TOKEN=你的扣子API令牌
COZE_BOT_ID=7573579561607331840
NODE_ENV=production
PORT=3014
```
