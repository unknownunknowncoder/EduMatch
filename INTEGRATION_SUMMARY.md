## 🚀 使用流程

### 1. 配置API（一次性）
```bash
# Windows
setup-coze-api.bat

# macOS/Linux  
./setup-coze-api.sh
```

### 2. 启动项目
```bash
npm run dev
```

### 3. 使用功能
1. 访问首页
2. 输入学习关键词（如：Python编程、线性代数）
3. 点击"AI推荐"
4. 查看推荐结果
5. 点击"立即学习"跳转到资源
